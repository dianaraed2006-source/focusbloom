# ابدئي من هنا — FocusBloom 4.0

## ما تم تجهيزه
- الواجهة الكاملة.
- الخطة الدراسية التفاعلية.
- مواد المستخدم ومساحات الملفات.
- رفع ملفات محلي + رفع سحابي اختياري.
- Supabase Auth والمزامنة.
- قاعدة البيانات وRLS.
- Storage خاص بكل مستخدم.
- تلخيص المحاضرات.
- مساعد الصيدلة.
- التعرف على علبة الدواء.
- مخطط أسبوعي ذكي.
- Drug Explorer وopenFDA/RxNorm.

## الجزء الأول: رفع الموقع إلى GitHub
ارفعي ملفات الجذر إلى المستودع، خصوصًا:
- index.html
- app-v40.js
- styles-v40.css
- config.js
- cloud.js

لا تحذفي ملف إثبات ملكية Google.

## الجزء الثاني: قاعدة البيانات
1. افتحي Supabase.
2. SQL Editor.
3. New query.
4. افتحي الملف `supabase/FULL_SETUP.sql`.
5. انسخي كل محتواه والصقيه.
6. اضغطي Run.
7. يجب أن تظهر رسالة نجاح.

## الجزء الثالث: Edge Functions
أنشئي الوظائف التالية بالأسماء نفسها:
- pharmacy-ai
- medicine-vision
- course-summarizer
- weekly-study-planner

لكل وظيفة استخدمي ملف `index.ts` الموجود في مجلدها داخل:
`supabase/functions/`

ثم Deploy لكل وظيفة.

## الجزء الرابع: المفتاح
أنتِ أضفتِ بالفعل:
`OPENAI_API_KEY`
داخل Edge Function Secrets.

لا تضعيه في GitHub ولا في الموقع.

## الجزء الخامس: ربط الموقع
1. افتحي FocusBloom المنشور.
2. ادخلي إلى «الإعداد والتفعيل».
3. ضعي:
   - Project URL: رابط المشروع دون `/rest/v1`
   - Publishable key: المفتاح العام الذي يبدأ غالبًا بـ `sb_publishable_`
4. اضغطي «حفظ واختبار الاتصال».
5. انتقلي إلى «الحساب والمزامنة».
6. أنشئي حسابًا أو سجلي الدخول.

## الاختبار النهائي
- أضيفي مادة من الخطة.
- افتحي المادة.
- ارفعي ملفًا.
- اختاري رفع نسخة سحابية.
- افتحي محلل المواد.
- استخرجي النص.
- اختاري «حزمة كاملة».
- اضغطي إنشاء الحزمة.
