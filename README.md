# FocusBloom

موقع دراسة كامل يعمل مباشرة على GitHub Pages.

## المزايا
- Pomodoro ومؤقت عادي
- إدارة المواد
- المهام
- مخطط شهري
- إحصائيات
- إنجازات
- نبتة تنمو حسب الدراسة
- ملف شخصي
- الوضع الداكن
- حفظ البيانات في المتصفح عبر Local Storage

## النشر
ارفعي الملفات التالية إلى GitHub:
- index.html
- styles.css
- app.js

ثم فعّلي GitHub Pages من Settings > Pages > Deploy from a branch > main / root.

## ملاحظة
هذه النسخة لا تحتوي على تسجيل دخول أو قاعدة بيانات سحابية. البيانات محفوظة على نفس الجهاز والمتصفح.
